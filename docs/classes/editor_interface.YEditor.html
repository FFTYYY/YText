  • Inherited from Renderer.get_renderer

    确定一个渲染器。

    Parameters

    • nodetype: NodeType

      节点类型。如果 stylename 为 undefined 则必须为 text 或 paragraph，否则必须为 inline、group、struct、support 之一。

    • stylename: string = undefined

      样式名。如果为 undefined 就表示无样式（使用默认渲染器）。

    Returns EditorRenderer_Func

    如果 stylename 是 undefined 或者没有找到渲染器，就范围这个节点类型的默认渲染器，否则返回找到的渲染器。
